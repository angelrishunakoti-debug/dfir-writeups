# Pre Security — Offensive vs Defensive Security (TryHackMe)

## Key Takeaway
- **Offensive Security:** simulating attacks (pentesting, red teaming) 
  to find weaknesses before real attackers do.
- **Defensive Security:** monitoring, detecting, and responding to 
  attacks (SOC, DFIR, blue team) — this is my career track.

## Roles Covered
| Offensive | Defensive |
|---|---|
| Penetration Tester | SOC Analyst |
| Red Teamer | DFIR Analyst |
| Bug Bounty Hunter | Threat Hunter |

## Why This Matters for Me
Confirms my focus: DFIR sits fully under Defensive Security. 
I'll build offensive-concept knowledge (attacker techniques via 
MITRE ATT&CK) only to support detection/investigation — not as 
a career path itself.
